Installation Instructions

Windows

  1. Download the .exe installer
  2. Run the installer
  3. Follow the installation wizard
  4. Launch SynapseNet from the Start menu or desktop shortcut

macOS

  1. Download the .dmg file
  2. Open the downloaded file
  3. Drag SynapseNet to your Applications folder
  4. Launch SynapseNet from Applications
  5. If prompted about an unidentified developer, go to System Preferences → Security & Privacy and click "Open Anyway"

Linux

  1. Download the .AppImage file
  2. Make it executable: chmod +x SynapseNet-1.0.0-linux.AppImage
  3. Run it: ./SynapseNet-1.0.0-linux.AppImage

System Requirements

  • Operating System: Windows 10+, macOS 10.15+, or Linux with GTK 3.0+
  • RAM: 2GB minimum, 4GB recommended
  • Disk Space: 500MB for application + storage for grains
  • Network: Internet connection (1 Mbps minimum, 10 Mbps recommended)
