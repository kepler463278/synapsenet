Getting Started

Installation

Download the appropriate version for your operating system from the download page and follow the installation instructions.

First Run

  1. Launch SynapseNet
  2. Click "Start Node" to connect to the network
  3. Wait for initial synchronization (usually 1-2 minutes)
  4. You're now part of the network!

Adding Knowledge

  1. Go to the "Add" tab
  2. Enter your knowledge, insights, or ideas
  3. Click "Add to Network"
  4. Your grain will be broadcast to peers and you'll earn NGT

Searching Network

  1. Go to the "Knowledge" tab
  2. Enter your search query
  3. Results will show grains from both local storage and network peers
  4. Each result shows confidence score and source

Architecture

Overview

SynapseNet is a peer-to-peer network with no central servers. Each node:

  • Stores grains locally
  • Connects directly to other nodes
  • Participates in swarm consensus
  • Earns rewards for contributions

Components

  • P2P Network: Direct peer-to-peer connections using libp2p
  • Storage: Local-first storage with SQLite
  • AI Layer: Embedding-based semantic search and reasoning
  • Swarm Intelligence: Distributed consensus mechanism
  • PoE System: Proof-of-Emergence reward calculation

P2P Protocol

Connection

Nodes discover each other through:

  • Bootstrap nodes (initial connection)
  • DHT (Distributed Hash Table)
  • Peer exchange

Message Types

  • Grain Broadcast: Share new knowledge
  • Query Request: Search for information
  • Swarm Formation: Create collective intelligence
  • Validation: Confirm grain quality

PoE Economics

How Rewards Work

Network Growth Tokens (NGT) are earned through Proof-of-Emergence:

Novel Grains

Reward = base_reward × novelty_score × network_multiplier

  • base_reward: 0.1 NGT
  • novelty_score: 0.0 to 10.0 (from semantic analysis)
  • network_multiplier: 1.0 + (peers / 1000)

Validation

0.01 to 0.1 NGT per validation, based on grain importance

Query Answers

0.05 to 0.5 NGT per answer, based on query complexity

Swarm Consensus

0.1 to 2.0 NGT per participation, based on swarm size and complexity

Important Notes

  • NGT cannot be bought or sold
  • NGT can only be earned through contribution
  • NGT represents recognition of value, not speculation
  • No blockchain, no mining, no staking

FAQ

Is my data private?

Yes. All data is stored locally on your device. When you share grains, you choose what to share. The network cannot access your private data.

Do I need to keep the node running?

No. You can start and stop the node anytime. When offline, you can still search local grains and view your rewards.

How much disk space does it use?

The application is ~100MB. Each grain is typically 1-10KB. 1000 grains = ~10MB.

Can I run multiple nodes?

Yes, but each node should have its own data directory. Running multiple nodes from the same device doesn't increase rewards.

What if I lose my data?

Your grains are stored locally. Back up your data directory regularly. Grains you've shared are also stored by peers who received them.

Troubleshooting

Node won't start

  • Check your internet connection
  • Ensure ports 9900-9910 are not blocked
  • Try restarting the application

No peers connecting

  • Check firewall settings
  • Wait a few minutes for peer discovery
  • Ensure you're not behind a restrictive NAT

Search returns no results

  • Ensure node is running and connected
  • Try different search terms
  • Wait for network synchronization
