## 🔧 Technical Stack

### Backend
- **Language:** Rust 🦀
- **Async Runtime:** Tokio
- **P2P:** libp2p
- **Storage:** SQLite + Parquet
- **Crypto:** Ed25519, Dilithium, Kyber
- **AI:** ONNX Runtime

### Frontend
- **Framework:** React + TypeScript
- **Build Tool:** Vite
- **Desktop:** Tauri 2.0
- **Styling:** CSS Modules

### Testing
- **Unit Tests:** Rust built-in
- **Integration:** Tokio test
- **UI Tests:** Vitest + Testing Library
- **Benchmarks:** Criterion
